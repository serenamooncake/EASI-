# 皮肤评分计算器 - 部署指南

## 项目简介

这是一个静态网站，包含 HTML、CSS 和 JavaScript 文件，支持中文界面。

---

## 方案一：GitHub Pages（免费，推荐）

### 步骤：

1. **注册 GitHub 账号**
   - 访问 https://github.com
   - 注册一个免费账号

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名称：例如 `easi-scorad-cn`
   - 设置为 Public
   - 点击 "Create repository"

3. **上传文件**
   - 点击 "uploading an existing file"
   - 将项目文件夹中的所有文件拖拽到上传区域
   - 提交并推送到仓库

4. **启用 GitHub Pages**
   - 进入仓库的 "Settings"
   - 找到 "Pages" 部分
   - 在 "Source" 中选择 "main" 分支
   - 点击 "Save"
   - 等待几分钟，网站将在 `https://您的用户名.github.io/easi-scorad-cn/` 上线

---

## 方案二：Vercel（免费，自动化）

### 步骤：

1. **注册 Vercel 账号**
   - 访问 https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "New Project"
   - 选择您的 GitHub 仓库
   - 点击 "Import"

3. **部署**
   - 保持默认设置
   - 点击 "Deploy"
   - 几秒钟后，您的网站将上线！

---

## 方案三：Netlify（免费，拖拽上传）

### 步骤：

1. **注册 Netlify 账号**
   - 访问 https://netlify.com
   - 注册并登录

2. **部署**
   - 进入 "Sites" 页面
   - 找到 "Add new site" → "Deploy manually"
   - 将整个项目文件夹拖拽到上传区域
   - 等待上传完成，网站立即上线！
   - 您可以在设置中自定义网站名称

---

## 方案四：云服务器部署（适合生产环境）

如果您有阿里云、腾讯云等服务器：

### Nginx 配置示例：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    root /var/www/easi-webpage;
    index index.html;
    
    location / {
        try_files $uri $uri/ =404;
    }
}
```
