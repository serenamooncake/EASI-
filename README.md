# 皮肤评分计算器

这是一个用于特应性皮炎等皮肤疾病评分计算的网页工具。

## 功能

- **SCORAD** - 特应性皮炎评分指数
- **EASI** - 特应性皮炎面积和严重程度指数
- **vIGA-AD™** - 研究者整体评估
- **DLQI** - 皮肤病生活质量指数
- **NRS** - 瘙痒数字评分量表
- **POEM** - 特应性皮炎患者导向湿疹指标
- **ADCT** - 特应性皮炎控制工具
- **PCT** - 瘙痒症控制工具
- **T2** - 2型炎症性疾病共病筛查

## 语言支持

- 中文
- 英文
- 西班牙语
- 法语
- 荷兰语
- 等

## 部署

可直接部署到 Vercel 或 Netlify 等静态网站托管平台。
